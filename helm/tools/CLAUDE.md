# helm/tools/ - Tool Server Charts

Helm charts for deploying MCP tool servers that provide capabilities to agents.

## Charts

| Directory | Tool |
|-----------|------|
| `grafana-mcp/` | Grafana MCP server (query and visualization tools) |
| `querydoc/` | QueryDoc MCP server (documentation query tools) |

Each chart contains: `Chart.yaml`, `Chart-template.yaml`, `values.yaml`, `templates/`.
