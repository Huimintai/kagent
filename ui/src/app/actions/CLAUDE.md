# ui/src/app/actions/ - Server Actions

Next.js server actions for backend API calls.

| Directory | Purpose |
|-----------|---------|
| `api/` | API action modules (has `auth/` subdir for OAuth flows) |

## Top-Level Action Files

| File | Purpose |
|------|---------|
| `agents.ts` | Agent CRUD operations |
| `auth.ts` | Authentication helpers |
| `feedback.ts` | User feedback submission |
| `memories.ts` | Memory management |
| `modelConfigs.ts` | Model config operations |
| `models.ts` | Model listing |
| `namespaces.ts` | Namespace queries |
| `promptTemplates.ts` | Prompt template CRUD |
| `providers.ts` | Provider listing |
| `scheduledRuns.ts` | Schedule management |
| `servers.ts` | Server operations |
| `sessions.ts` | Session management |
| `tools.ts` | Tool operations |
