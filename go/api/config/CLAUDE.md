# go/api/config — Build and CRD Configuration

## Structure

| Directory | Role |
|-----------|------|
| `crd/` | CRD manifest configuration |
| `crd/bases/` | Base CRD YAML manifests (generated by controller-gen) |

## Usage

CRD base manifests in `crd/bases/` are generated from Go type definitions in `v1alpha2/` using `controller-gen`. They are consumed by Helm charts for installation.
