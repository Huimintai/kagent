# cli/internal/common/

Shared utility packages used across CLI commands and generators.

## Subdirectories

| Directory | Purpose |
|-----------|---------|
| `exec/` | Shell command execution helpers |
| `fs/` | Filesystem operations (directory creation, file writing) |
| `generator/` | Base template rendering engine for project scaffolding |
| `image/` | Container image reference parsing and construction |
| `k8s/` | Kubernetes client helpers (context, namespace detection) |
| `prompt/` | Interactive terminal prompts for user input |
