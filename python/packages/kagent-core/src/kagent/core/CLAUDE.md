# kagent.core - Core Library Implementation

## Subpackages

| Directory | Purpose |
|-----------|---------|
| `a2a/` | A2A protocol utilities (config, constants, HITL helpers, task store, requests) |
| `tracing/` | OpenTelemetry integration (span processor, tracing utilities) |

## Top-Level Modules

| Module | Role |
|--------|------|
| `_config.py` | Configuration loading |
| `_logging.py` | Structured logging setup |
