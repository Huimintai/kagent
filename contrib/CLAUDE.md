# contrib/ - Community Contributions & Extensions

Additional resources, integrations, and documentation not part of the core platform.

## Sub-packages

| Directory | Role |
|-----------|------|
| `addons/` | Kubernetes addon manifests (Prometheus, Grafana, metrics-server) |
| `cncf/` | CNCF compliance documents (technical review, security self-assessment) |
| `memory/` | Memory backend configurations (Qdrant) |
| `tools/` | External MCP server tool integrations |
